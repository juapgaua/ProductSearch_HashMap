# Sistema de Busca de Produtos em Java (Armazenamento Associativo)

Este projeto foi desenvolvido como atividade prática da disciplina **Algoritmos, Estrutura de Dados e TAD**, com foco no uso de **armazenamento associativo** utilizando a estrutura de dados **HashMap** em Java.

O objetivo do sistema é simular um módulo de e-commerce capaz de:

- cadastrar produtos  
- buscar produtos por código  
- remover produtos  
- listar todos os produtos  
- gerenciar os dados de forma eficiente usando chave-valor  

O programa é totalmente **modularizado**, **interativo**, **didático**, e ideal para quem está iniciando na linguagem Java e na compreensão de estruturas de dados.

## Conceitos Aplicados

Este projeto utiliza:

- **Armazenamento Associativo (Mapas)**
- **HashMap como estrutura de dados**
- **Operações chave → valor**
- **Busca em tempo constante O(1)**
- **Modularização de código**
- **Menu interativo**
- **Boas práticas de programação**

## Estrutura do Projeto

/projeto  
├── Produto.java //Classe que representa um produto do e-commerce.  
├── CadastroProdutos.java //Responsável por armazenar, buscar, listar e remover produtos do HashMap.  
└── Main.java //Contém o menu interativo e a lógica de interação com o usuário.

## ▶ Funcionalidades

- **Cadastrar produto**
- **Buscar produto por código**
- **Remover produto**
- **Listar todos os produtos**
- **Menu simples e intuitivo**
- **Sistema modular e fácil de entender**

## Como Executar

### 1. Compile os arquivos

No terminal, dentro da pasta do projeto:

```bash
javac *.java
```

2. Execute o programa
```bash
java Main
```
